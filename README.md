# Bike sales Data Cleaning 

This repository contains an Excel sheet where raw expense data was cleaned and prepared for analysis.

## Data Cleaning Steps
- Removed duplicate records to ensure accuracy.
- Handled missing values by either imputing or removing incomplete rows.
- Corrected typos and standardized category names.
- Converted dates into a consistent format (YYYY-MM-DD).
- Normalized currency values for consistency.
- Removed irrelevant columns that were not needed for analysis.
- Created a pivot table to summarize expenses by category
- Built an interactive dashboard using pivot charts and slicers.
- Added slicers for region and education to enable dynamic filtering

## Outcome
The cleaned dataset is now consistent, accurate, and ready for further analysis or visualization.
